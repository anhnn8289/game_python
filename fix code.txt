Bug 1:mình phát hiện một lỗi tiềm ẩn có thể gây crash chương trình khi khối gạch (piece) vượt ra ngoài biên của bảng (board) (khi không kiểm tra kỹ giới hạn mảng)
Nếu self.current_pos["y"] + y hoặc self.current_pos["x"] + x vượt quá kích thước của self.board, chương trình sẽ bị lỗi IndexError.

	code lỗi:
		def get_current_board_state(self):
			board = [x[:] for x in self.board]
			for y in range(len(self.piece)):
				for x in range(len(self.piece[y])):
					board[y + self.current_pos["y"]][x + self.current_pos["x"]] = self.piece[y][x]
			return board
	
	Fix lỗi:
		def get_current_board_state(self):
			board = [x[:] for x in self.board]
			for y in range(len(self.piece)):
				for x in range(len(self.piece[y])):
					board_y = y + self.current_pos["y"]
					board_x = x + self.current_pos["x"]
					if 0 <= board_y < self.height and 0 <= board_x < self.width:
						board[board_y][board_x] = self.piece[y][x]
			return board
